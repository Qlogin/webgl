WebGL Examples and Demos!

Here you can find example and demos of three-dimensional interactive applications that used WebGL. These applications are independent of the particular platform and can be run like any other Web application within the latest web browsers. Enjoy 3D Computer Graphics developed in JavaScript!

1. Tessellation and Twist

First program allow you to demonstrate twist and tessellation. User can enter both the number of subdivisions and the angle. Rotation is implemented in the vertex shader despite of doing the same operation in JavaScript code. Simple shading based on distance from center attenuation.

View details »

2. Painting with the mouse

This application will let you created what look like curves by connecting a series of vertices with line segments. The vertices are generating through the mouse move event while a mouse button is depressed. A new curve is started each time the mouse button is depressed. User can change the drawing color and line width.

View details »

3. Geometric CAD

CAD-like application that gives you some experience with transformations, 3D-objects and interaction. A simple modeling program that lets the user populate the canvas with spheres, cones, and cylinders. All the information - choice of an object, its size, its location and its orientation - is entered interactively. But this is only the sketch of usefull graphic interface (that implemented in next example).

4. CAD with Lighting

This is an extension of the previous example. Here we get a more realistic image by using lighting. User can assign material properties to objects and can control light sources. Also user interface has been improved: camera movement using mouse, interactive object placement, immediate reaction on parameters changing
